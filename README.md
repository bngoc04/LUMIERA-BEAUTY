## Prerequisites
- Node.js installed.

## Setup
1. Open a terminal in the project folder.
2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Project
This project has a **Backend** (API) and a **Frontend** (HTML/JS). Both components must be active.

### 1. Start the Backend Server
Run the following command in your terminal:
```bash
node server.js
```
*The server will start at `http://localhost:3000`.*

### 2. Open the Website
Open `homepage.html` in your browser (simply double-click the file, or use "Live Server" in VS Code).
